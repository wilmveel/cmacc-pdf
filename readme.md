# Introduction

Command line tool to generate pdf files.

# Install

```
npm install -g cmacc-pdf
```

# Usage

```
cmacc-pdf <input file> <output file>
```

## Example

```
cmacc-pdf ./hello_world.cmacc 
```
output: ./hello_world.pdf

```
cmacc-pdf ./hello_world.cmacc ./hello.pdf
```
output: ./hello.pdf
